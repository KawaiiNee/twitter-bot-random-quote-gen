# vite - react eslinting setup

**install eslint**
`npm install --save-dev eslint-config-react-app eslint@^8.0.0`
`{"extends": ["react-app", "react-app/jest"]} add this to .eslintrc.json`

### npm i vite-plugin-eslint

`Add to vite.config.js as plugin`
